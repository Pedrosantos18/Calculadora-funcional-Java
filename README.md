Projeto de calculadora de linha de comando desenvolvida em Java, utilizando expressões lambda e funções funcionais.
O objetivo é praticar conceitos de Programação funcional, tratamento de excenções, e armazenamento de histórico de operações.

Funcionalidades:
  - Operações básicas: adição, subtração, multiplicação e divisão
  - Funções extras: exponenciação e raiz quadrada.
  - Histórico das operações realizadas.
  - Tratamento de erros e entradas inválidas.
  - Uso de lambdas para definir operações matemáticas.
    
Tecnologias Utilizadas:
  - Java 24
  - Paradigma funcional (Lambdas, Function, BiFunction)
  - Tratamento de exceções
  - Coleções (List) para histórico

Conceitos Praticados:
  -Expressões Lambda e Interfaces Funcionais
  -Programação Funcional em Java
  -Tratamento de Exceções (try-catch)
  -Coleções (ArrayList)
  -Entrada e saída com Scanner
